# SevaMarket - ISKCON Community Marketplace

**SevaMarket** (sevamarket.com) - A multi-vendor marketplace platform for ISKCON congregation members to offer their products and services to the devotee community and supporters.

> *"Seva" = Selfless Service | "Market" = Marketplace*

## Vision

Create a trusted marketplace where ISKCON devotees can:
- **Vendors**: Showcase and sell their products/services
- **Buyers**: Support community members by purchasing from verified devotee-owned businesses
- **Community**: Strengthen economic bonds within the ISKCON family

---

## Project Overview

| Aspect | Details |
|--------|---------|
| Platform | Shopify |
| Multi-Vendor App | Webkul Multi Vendor Marketplace |
| Initial Vendors | 5 (scalable to 25+) |
| Offerings | Products + Services |
| Payment Model | Centralized (marketplace collects, pays vendors) |
| Vendor Approval | Required (admin approves each vendor) |
| Target Audience | ISKCON congregation & supporters |

## Technology Stack

### Platform
- **Shopify** - E-commerce platform ($29-79/month)
- **Webkul Multi Vendor Marketplace** - Multi-vendor functionality (~$30-50/month)

### Features Needed
- Vendor registration & approval workflow
- Vendor dashboard for product/service management
- Commission management (marketplace takes %)
- Centralized payment processing
- Order routing to correct vendor
- Vendor payouts (weekly/monthly)
- Reviews & ratings per vendor

## Quick Start

### Prerequisites
- Shopify account
- Domain name
- Payment processor (Shopify Payments or PayPal)

### Installation Steps

1. **Create Shopify Store**
   ```
   Go to: https://www.shopify.com/
   Sign up for Basic plan ($29/month)
   ```

2. **Install Multi-Vendor App**
   ```
   Shopify Admin → Apps → Search "Webkul Multi Vendor"
   Install and configure
   ```

3. **Clone This Repository**
   ```bash
   git clone https://github.com/rrajeshprabhu/sevamarket.git
   cd sevamarket
   ```

4. **Follow Setup Guides**
   - See `docs/MARKETPLACE-SETUP.md` for admin setup
   - See `docs/VENDOR-GUIDE.md` for vendor onboarding
